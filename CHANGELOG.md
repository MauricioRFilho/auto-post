# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-26

### Added
- **Backend**: NestJS API with Links, Products, and Posts modules.
- **Scraper**: Python Playwright worker with Mercado Libre support.
- **Dashboard**: Next.js 14 interface with shadcn/ui.
- **Automation**: n8n workflow for multi-channel posting (Instagram, Pinterest, WhatsApp).
- **Core**: Docker and Docker Compose infrastructure.
- **Open Source**: Added LICENSE, Code of Conduct, Contributing guidelines, and Security policy.
