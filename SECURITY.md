# Security Policy

## Supported Versions

Currently, only the latest version of Cadence Auto-Post is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to [mauricio.srfh@gmail.com]. All security vulnerabilities will be promptly addressed.

Please include the following in your report:
- Type of issue (e.g., buffer overflow, SQL injection, RCE)
- Full paths of source file(s) related to the manifestation of the issue
- The location of the affected code
- A description of the issue
- Steps to reproduce

# Política de Segurança

## Versões Suportadas

No momento, apenas a versão mais recente do Cadence Auto-Post é suportada com atualizações de segurança.

| Versão | Suportado          |
| ------ | ------------------ |
| 1.0.x  | :white_check_mark: |

## Relatando uma Vulnerabilidade

Se você descobrir uma vulnerabilidade de segurança neste projeto, envie um e-mail para [mauricio.srfh@gmail.com]. Todas as vulnerabilidades de segurança serão tratadas prontamente.

Por favor, inclua o seguinte em seu relatório:
- Tipo de problema (ex: buffer overflow, SQL injection, RCE)
- Caminhos completos dos arquivos fonte relacionados à manifestação do problema
- A localização do código afetado
- Uma descrição do problema
- Passos para reproduzir
