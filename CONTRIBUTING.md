## 🇧🇷 Português (PT-BR)

### 🚀 Como Contribuir
1. **Fork** o repositório.
2. Crie uma **Branch** para sua funcionalidade (`git checkout -b feature/nova-funcao`).
3. Siga os padrões de **Clean Code**.
4. Abra um **Pull Request** detalhando suas mudanças.

### 📜 Regras de Código
- Utilize **Português (PT-BR)** para comentários e documentação interna.
- Siga as regras de **StyleCop** (SA1512, SA1413, SA1515).
- Use `is null` em vez de `== null` para checagens de nulo.
