## 🇧🇷 Português (PT-BR)

### 1. Painel de Controle (Dashboard)
O Dashboard é a sua central de comando.
- **Inserir Link**: Cole o link do Mercado Livre, Shopee ou Magalu diretamente no formulário.
- **Status**: Verifique se o post foi enviado com sucesso no Telegram.
- **Configurações**: Altere seu Token do Telegram em tempo real.

### 2. Configurando o Telegram
Você precisa de duas informações: **Bot Token** e **Chat ID**.
1. **Bot Token**: Fale com o [@BotFather](https://t.me/botfather) no Telegram, crie um novo bot e copie o token gerado.
2. **Chat ID**: Crie seu canal de ofertas, adicione seu bot como **Administrador** e envie uma mensagem qualquer. Depois use um bot como o [@userinfobot](https://t.me/userinfobot) para descobrir o ID do canal (lembre-se que IDs de canais costumam começar com `-100`).

### 3. Gerenciando Links
- **Links Ativos**: Os links que você adiciona entram em uma fila para o scraper.
- **Scraping**: O sistema extrai automaticamente o título, preço original, preço em oferta e a imagem do produto.

## 🇺🇸 English (EN)

### 1. Control Panel (Dashboard)
The Dashboard is your command center.
- **Insert Link**: Paste the Mercado Livre, Shopee, or Magalu link directly into the form.
- **Status**: Check if the post was successfully sent on Telegram.
- **Settings**: Change your Telegram Token in real-time.

### 2. Configuring Telegram
You need two pieces of information: **Bot Token** and **Chat ID**.
1. **Bot Token**: Talk to [@BotFather](https://t.me/botfather) on Telegram, create a new bot, and copy the generated token.
2. **Chat ID**: Create your offers channel, add your bot as an **Administrator**, and send any message. Then use a bot like [@userinfobot](https://t.me/userinfobot) to find the channel ID (remember that channel IDs usually start with `-100`).
